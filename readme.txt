Пояснительная записка к проекту “Морской бой”.

Данная программа имитирует игру в морской бой человека с компьютером. 

Вначале игры нужно расставить свои корабли на левом поле мышкой. После того, как корабль поставлен, нужно нажать Enter. Когда все корабли будут расставлены, на экране будет указано, кто ходит первым. Результаты выстрелов противника показываются на левом поле, ваших выстрелов - на правом. Для того, чтобы выстрелить, нужно мышкой нажать по клетке на правом поле. 
Обозначения: черный - мимо, красный - попал, синий - клетки, где не может быть кораблей, зеленый - клетки, на которых располагались корабли противника.

Для выхода из игры нужно нажать Esc.

Для работы программы требуется установленная библиотека PyQt5.
